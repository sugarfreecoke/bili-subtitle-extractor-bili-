# 🎬 B站字幕批量提取器（轻量版）

输入 UP 主 UID 或单个视频链接，一键下载所有视频的 **中文字幕**，以视频标题命名保存为 `.txt` 文件，方便后续 AI 分析、知识库构建。

## 🛠️ 安装与启动

### 1️⃣ 安装依赖

在项目目录下打开终端（PowerShell / CMD），执行：

```bash
npm install
```

只需安装 `axios` 一个依赖包，**几秒钟**即可完成。

### 2️⃣ 启动服务

```bash
node server.js
```

启动后，终端会显示服务地址，并**自动打开浏览器**访问 `http://localhost:5566`。

**如果端口被占用**：
- 修改 `server.js` 中的 `PORT` 变量（如改为 `8899`），重新启动。
- 或先终止占用端口的进程：`taskkill /f /im node.exe`

### 3️⃣ 开始使用

1. 在浏览器界面中填写：
   - **UP 主 UID**（数字，如 `546195`）或 **视频链接**（如 `https://www.bilibili.com/video/BV1xx...`）
   - **Cookie**（从浏览器 F12 复制，见下方说明）
   - 可选：限制处理视频数量（`0` 表示全部）

2. 点击 **「开始提取」**，等待运行完成。

3. 提取完成后，点击 **「📂 打开输出文件夹」** 即可查看字幕文件（保存在 `subtitles/` 目录）。

## 🍪 如何获取 Cookie

1. 在 **Chrome / Edge** 浏览器中打开 [B站](https://www.bilibili.com) **并已登录账号**。
2. 按 **F12** 打开开发者工具。
3. 点击顶部 **「网络」(Network)** 标签。
4. **刷新页面**（按 F5）。
5. 在左侧请求列表中，**点击第一个请求**（通常是 `www.bilibili.com`）。
6. 在右侧面板找到 **「请求头」(Request Headers)**，找到 **`Cookie:`** 这一行。
7. **右键 → 复制值**，或选中完整 Cookie 文本（很长的一串），按 `Ctrl+C` 复制。
8. 将复制的内容粘贴到工具的 Cookie 输入框中。

> ⚠️ Cookie 相当于你的登录凭证，**请勿分享给他人**。本工具仅在本地使用，不上传任何数据。

## ❓ 常见问题

### Q1：启动时提示端口被占用（EADDRINUSE）
- 关闭当前终端，修改 `server.js` 中的 `PORT` 变量为其他数字（如 `8899`），保存后重新启动。
- 或使用 `taskkill /f /im node.exe` 终止所有 Node 进程后再启动。

### Q2：访问界面显示 404
- 确认 **`public/index.html`** 文件是否存在且内容完整。
- 如果缺失，手动创建 `public` 文件夹，将完整 HTML 代码保存为 `index.html`（代码见项目文件）。

### Q3：提取速度很慢
- 每个请求之间有 500ms 延迟，避免被 B 站限流。大量视频需耐心等待。

### Q4：某个视频出现「无字幕」
- 说明该视频没有内置字幕（AI 字幕或 CC 字幕）。后续版本可能集成语音转写（Whisper）。

### Q5：Cookie 过期了怎么办？
- 重新从浏览器 F12 复制最新 Cookie 即可，无需重启服务。

### Q6：可以一次处理多个 UP 主吗？
- 目前一次只能处理一个 UP 主。如需处理多个，请完成后重新输入新的 UID 并再次点击提取。
